Info

This save bank is based on earth's save archive

After pastebin started deleting paste of save file (rip save archive), this save bank was created to provided save files for players who lost their save files, and to store them more securely and permanently.

Contributing

If you wish to donate your save file, DM me (Buck#4437) on discord and I'll see if I can find a spot for it. If you have encountered any issues, DM me or ping me on AD discord.
