Info

This save bank is based on earth's save archive

After pastebin started deleting paste of save file (rip save archive), this save bank was created to continue providing save files for players who lost their save files, and to store them more permanently.

FAQ

Why are custom saves removed?

1. It is rarely used, as there are better alternatives (such as google docs)
2. The cost of maintainance is high.

I have a save file. How can I submit it to save bank?

If you wish to donate your save file, DM me (Buck#4437) on discord and I'll see if I can find a spot for it.
